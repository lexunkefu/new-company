# TechCorp 公司网站

基于 Next.js 的现代化静态公司网站。

## 功能特性

- 🚀 基于 Next.js 14 和 React 18
- 📱 响应式设计，移动端友好
- ⚡ 快速的页面加载性能
- 🔍 SEO 友好
- 🎨 现代化的 UI 设计
- 📄 完整的页面模块

## 页面模块

1. **产品展示** - 展示公司产品和解决方案
2. **下载中心** - 提供软件下载和文档
3. **客服中心** - 技术支持与帮助文档
4. **视频中心** - 产品演示和教程视频
5. **联系我们** - 联系方式和反馈表单

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
