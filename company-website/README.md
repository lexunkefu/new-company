# TechCorp 公司网站

基于 Next.js 14 的现代化静态公司网站，包含产品展示、下载中心、客服中心、视频中心和联系我们等完整功能。

## 功能特性

- 🚀 基于 Next.js 14 和 React 18
- 📱 响应式设计，移动端友好
- ⚡ 静态生成，快速加载
- 🎨 现代化 UI 设计，使用 Tailwind CSS
- 📄 完整的页面模块
- 🔍 SEO 优化
- 🌐 支持多语言（中文）

## 页面模块

1. **首页** - 公司介绍和核心功能展示
2. **产品展示** - 展示公司产品和解决方案
3. **下载中心** - 提供软件下载和文档
4. **客服中心** - 技术支持与帮助文档
5. **视频中心** - 产品演示和教程视频
6. **联系我们** - 联系方式和反馈表单

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- React Icons
- date-fns
